# Sport

Лендинг с выбором тарифов для фитнес‑продукта: пользователь видит карточки подписок, акционные цены, юридический блок и форму согласия.

## Что реализовано

- Страница выбора тарифа с адаптивной версткой
- Карточки тарифов с состояниями `hit`, скидкой и отображением полной/акционной цены
- Загрузка тарифов с API через серверный прокси-роут
- Таймер оффера и обновление отображения цен
- Скелетон-лоадер во время загрузки данных

## Технологии

- Next.js (App Router)
- React
- TypeScript
- SCSS Modules
- Tailwind CSS (utility-классы в UI)
- Zustand (локальное состояние, таймер)
- Fetch API / route handlers (`app/api`) для работы с данными
